## Hi, I am Shubham

I build AI-assisted tools, research workflows, and practical full-stack apps. My repos usually sit at the intersection of Python systems, TypeScript interfaces, data workflows, and LLM-backed automation.

- Currently exploring: fact-checking systems, autonomous research, spaced repetition, and synthetic-data benchmarks.
- Favorite build zone: taking a rough research/product idea and making it usable enough to test with real people.
- Working style: ship small, measure what matters, keep the interface clean, and let the data tell the truth.
